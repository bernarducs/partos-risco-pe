### Dashboard da produção de Partos (Normais e de Risco) em Pernambuco

```
# clonando...

$ git clone https://github.com/bernarducs/partos-risco-pe.git
$ cd partos-risco-pe
$ python -m venv ve; source ve/bin/activate; python install -r requirements.txt


# rodando...

$ python app/main.py

```
